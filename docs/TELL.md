# TELL

**Usage:**  
TELL STORY

**Description:**  
Makes the actor tell **The story** to the player.

**Parameters:**
- `STORY` â€“ The story to tell  

**Notes:**
- Only **one parameter** is allowed.  
- Autocomplete will suggest example stories, they may not exist.  
- Using an unknown story name will not trigger red squiggles, but may cause runtime errors if the resource does not exist.
- This command blocks certain actions USE CAREFULLY
- DO NOT loop stories with one another

**Example:**
TELL ANGRY_GUY

-Makes the actor tell the story **ANGRY_GUY**.
